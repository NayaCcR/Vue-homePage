# Vue homePage
一个基于[Vue Argon](https://www.creative-tim.com/product/vue-argon-design-system) 设计的个人主页

## 先决条件
[nodejs](https://nodejs.org/)  
需要node v14.19以下，sass^4.11.0不支持node v14.19以上版本  
推荐使用`yarn`包管理器

## 快速开始
如果您没有yarn包管理器
可以使用以下代码安装
```sh
npm install yarn -g
```

### 补全依赖
yarn  
```sh
yarn
```  

npm  
```sh
npm i
```

### 启动服务
yarn
```sh
yarn serve
```

npm
```sh
npm run serve
```
